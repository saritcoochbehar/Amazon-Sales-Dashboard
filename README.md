Excel Data Analytics Dashboard: Explore raw data with ease. Interactive charts, filters, and KPIs help uncover insights quickly without predefined models. A simple, flexible tool for business analysis and decision-making.

**Overview**
The **Excel Data Analytics Dashboard** is a simple, flexible tool designed to help users explore raw datasets without predefined models. It enables quick insights, trend discovery, and KPI tracking through interactive charts, filters, and clean visuals.

**Features**
- Interactive charts and graphs for data exploration  
- Dynamic filters to slice and dice datasets  
- KPI tracking for business performance monitoring  
- No predefined models ‚Äî discover insights freely  
- Easy-to-use interface built entirely in Excel
